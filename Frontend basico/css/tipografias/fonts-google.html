Tipografia com Google Fonts (Roboto, Merriweather, Montserrat e Source Code Pro)

Este exemplo demonstra como importar e aplicar famílias, pesos e tamanhos com uma escala tipográfica simples, além de boas práticas de fallback de fontes.

Títulos com Montserrat (sem serifa)

Montserrat é ótima para títulos modernos e limpos. Aqui usamos pesos 600 e 800 para dar hierarquia.

Texto corrido com Roboto (sem serifa)

Roboto é uma fonte de leitura confortável em interfaces. Você pode variar o peso com 300, 400, 500 e 700 para indicar hierarquia e ênfase.

Este parágrafo está com a classe .muted, que reduz contraste para textos auxiliares ou descrições longas.

Leitura longa com Merriweather (serifada)

Para blocos de leitura longa, fontes serifadas como Merriweather costumam funcionar bem, pois melhoram a fluidez visual em parágrafos densos. Ajustamos line-height e max-width para conforto.

Dica: controle a largura da coluna (em ch ou rem) e a altura de linha (1.6–1.9) para uma experiência mais agradável.

Trechos de código com Source Code Pro (monoespaçada)


/* Exemplo CSS */
:root {
  --font-sans: "Roboto", system-ui, -apple-system, "Segoe UI", Arial, sans-serif;
  --font-serif: "Merriweather", Georgia, "Times New Roman", serif;
  --font-title: "Montserrat", "Roboto", system-ui, Arial, sans-serif;
  --font-mono: "Source Code Pro", ui-monospace, SFMono-Regular, Menlo, Consolas, monospace;
}
