Tabela comparativa de propriedades de tipografia

Propriedade Valor Exemplo Descrição
font-family 'Times New Roman', serif Exemplo de família serifada Define a família/tipo de fonte a ser usada.
font-size 24px Tamanho de fonte maior (24px) Define o tamanho do texto (px, rem, em, etc.).
font-weight 700 (bold) Texto em negrito (700) Define o peso (espessura) da fonte (100–900).
font-style italic Texto em itálico Define estilo da fonte (normal, italic, oblique).
text-transform uppercase texto transformado para maiúsculas Transforma a capitalização (uppercase, lowercase, capitalize).
text-decoration underline Texto sublinhado Aplica decoração (underline, line-through etc.).
letter-spacing 3px Espaço entre letras ↑ Controla o espaçamento entre caracteres.
word-spacing 10px Espaço entre palavras ↑ Controla o espaçamento entre palavras.
line-height 2 Altura de linha maior Altura da linha; influencia legibilidade e respiro vertical.
text-align center Texto centralizado Alinha o texto (left, center, right, justify).
