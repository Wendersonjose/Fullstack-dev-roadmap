Tabela Comparativa

Método Como funciona Vantagens Desvantagens
Margin Auto margin: auto centraliza horizontalmente. Simples e direto. Não centraliza verticalmente.
Text-align + inline-block Pai usa text-align: center, filho vira inline-block. Funciona em layouts antigos. Não centraliza verticalmente, só horizontal.
Flexbox display: flex + justify-content + align-items. Centraliza em ambos os eixos facilmente. Precisa suporte de navegadores modernos (hoje todos suportam).
Grid display: grid + place-items: center. Mais simples que flex para centralizar. Exige CSS Grid (navegadores antigos não suportam).
Position Absoluta Define top/left: 50% + transform: translate. Centraliza em qualquer contexto. Menos semântico, pode dar problemas em layouts responsivos.
